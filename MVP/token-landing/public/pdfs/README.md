This directory contains pre-generated PDF files for download.

## Required Files

1. **joy-token-pitch-deck.pdf** - Pre-generated pitch deck presentation
2. **joy-token-whitepaper.pdf** - Pre-generated technical whitepaper

### Option 2: Manual Creation
Create professional PDF versions of:

### Option 3: Print to PDF
1. Open the pages in browser
2. Use browser's Print function (Ctrl/Cmd + P)
3. Select "Save as PDF"
4. Save to this directory
